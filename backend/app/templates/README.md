# Templates — domínio reservado

Esta pasta existe como **reserva conceitual** para a futura camada de Templates
(layouts visuais reutilizáveis por habilidades).

**Estado atual (Sprint 0):** vazia, sem código. Apenas Jinja2 em `ia/prompts/`
está ativo.

**Quando ativar:** somente quando ≥2 habilidades reutilizarem o mesmo layout
de saída (ex.: card de post para Instagram + LinkedIn). Antes disso, qualquer
abstração aqui é prematura.

Não adicionar classes, serviços, registros ou endpoints aqui antes dessa
necessidade concreta surgir. Ver §11 e §13 do plano arquitetural.
